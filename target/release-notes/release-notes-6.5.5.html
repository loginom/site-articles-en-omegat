Loginom 6.5.5 — описание изменений Исправлены ошибки в мастере настройки импорта из Excel-файла и экспорта в базу данных. Внесены исправления в визуализатор диаграмма. Исправлена ошибка в обработчике Цикл.

Fixes

Визуализатор Диаграмма: При использовании в качестве поля абсцисс столбца, содержащего только null-значения, возникала ошибка "Номер строки XXXXXXXXXX вне диапазона [0; 13]" (#9127).

Визуализатор Диаграмма: Обновлена библиотека, которая используется для построения диаграммы. В новой версии библиотеки были исправлены ошибки, из-за которых в некоторых случаях сбивался масштаб пузырьковой диаграммы (#9222).

Визуализатор Диаграмма: Исправлены ошибки сохранения параметров диаграммы, из-за которых могли не сохраняться настройки масштабирования (#9445).

Импорт из Excel-файла: Исправлено несколько ошибок, которые возникали на этапе настройки импорта из Excel-файла, при задании имени листа или именованного диапазона через переменную. Если пользователь завершал настройку игнорируя ошибки, то данные импортировались, но все поля имели переменный тип данных (#5790).

Импорт из Excel-файла: При использовании переменной в качестве значения параметра Выбор объекта после перехода ко второму шагу в мастере настройки возникала ошибка "Индекс листа Excel задан некорректно". Ошибку исправили (#9268).

Приложение: В мастерах настройки исправлена ошибка "Cannot read properties of null (reading 'setValue')", которая возникала при перемещении с помощью клавиш навигации в панели переключения страниц (#9399).

Экспорт в базу данных:В мастере настройки экспорта в БД для исходных столбцов отсутствовало контекстное меню. Ситуацию исправили (#8017).

Цикл: Исправлена ошибка Access violation, которая возникала, если на этапе синхронизации перед выполнением в базовом узле происходила ошибка (#9479).
